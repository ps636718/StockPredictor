# Stock Price Predictor

A Streamlit web application that predicts short-term stock price movements for NSE-listed equities using an ensemble of trained machine learning models and technical indicator-based feature engineering.

## Overview

This application fetches historical stock data via Yahoo Finance, computes a set of technical indicators and derived features, and generates day-wise price forecasts for a configurable horizon of up to 7 trading days. Predictions are visualized alongside historical price action using an interactive Plotly chart, with a confidence rating assigned to each forecasted day.

## Features

- Live data retrieval for any NSE-listed ticker via `yfinance`
- Automated feature engineering, including:
  - Price normalization (Close, High, Low, Open)
  - Simple and Exponential Moving Averages (SMA-7, EMA-12, EMA-26)
  - Relative Strength Index (RSI) and MACD
  - Lagged price features (1, 2, 3, and 7-day lags)
  - Daily return, price rate-of-change, and rolling volatility
  - Volume-based features, including log-transformed and relative volume
- Multi-day price prediction using pre-trained regression models (one model per forecast horizon day)
- Interactive historical vs. predicted price chart
- Day-wise prediction table with an associated confidence indicator
- Configurable prediction horizon (1 to 7 days) via the sidebar

   ```bash
   git clone <repository-url>
   cd stock-price-predictor
   ```

2. Create and activate a virtual environment (recommended):

   ```bash
   python -m venv venv
   source venv/bin/activate    # On Windows: venv\Scripts\activate
   ```

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Streamlit application from the project root:

```bash
streamlit run app.py
```

Once the app is running:

1. Enter an NSE stock ticker in the sidebar (e.g., `RELIANCE.NS`, `TCS.NS`, `INFY.NS`).
2. Select the desired prediction horizon (1–7 days).
3. Click **Predict** to generate the forecast chart and day-wise prediction table.

## Model Details

The application loads a dictionary of pre-trained models (`models.joblib`), keyed by forecast day (1 through 7). For a given input ticker:

1. The most recent 6 months of daily OHLCV data is fetched and enriched with the feature set listed above.
2. The latest feature row is scaled using the fitted `StandardScaler` (`scaler.joblib`).
3. Each day's model predicts a normalized closing price, which is then denormalized relative to the current price to produce an actual price forecast.

Confidence levels are assigned heuristically based on forecast horizon:

| Days Ahead | Confidence |
|------------|------------|
| 1–3        | High       |
| 4–5        | Medium     |
| 6–7        | Low        |

This reflects the general expectation that prediction accuracy degrades as the forecast horizon increases.
